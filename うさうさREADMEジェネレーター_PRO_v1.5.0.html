README ジェネレーター 使い方
入力すると、右側にREADMEがリアルタイム生成されます。完成したら📤から各形式で書き出し！（オフライン・自動保存）
1
🪪 プロジェクト情報を書く名前・一行説明・バージョン・ライセンス・作者。バッジもON/OFF。
2
🧩 テンプレートを選ぶ右上から「単一HTMLツール / ライブラリ / Webアプリ / ミニマル」を流し込み。
3
✍️ セクションを編集・追加
テキスト箇条書きコード表FAQ更新履歴📁 構成ツリー⚙️ 環境構築🌿 Git運用
見出し編集・▲▼並べ替え・ON/OFF・削除ができます。
4
🕒 フッターと 📦 GitHub構成 を設定作成日・最終更新・版・更新者（自動/手動・和暦対応）。雛形ZIP用のスタックや同梱ファイルもここで。
5
📤 出力する
MarkdownメモPDFCSVExcelWord🎁 雛形ZIP
または「⧉ MDコピー」でそのまま貼り付け。
💡 ヒント：右側は 👁プレビュー / 📝Markdown を切替できます。入力は自動保存。このガイドは右上 🛈 使い方 でいつでも開けます。
あとで見返すときは 🛈 使い方 から
📤 出力形式を選択
フッター（最終更新日時・版・最終更新者）は全形式に反映されます。PDFは印刷ダイアログから「PDFとして保存」を選択。Excel(.xls)／Word(.doc)はそれぞれExcel／Wordで開けます（オフライン時はバッジ画像が表示されない場合があります）。
